# 🎥 OBS Plugins for Linux (Arch)

Une petite collection de plugins **OBS Studio** compilés et maintenus pour Arch Linux.

Je maintiens ces plugins parce que :
- Les versions officielles arrivent parfois tard sur l'AUR
- J'aime avoir des versions propres et à jour
- J'adore bidouiller et simplifier la vie des autres manchots Linux 🐧

---

### 📦 Plugins disponibles

- **Media Playlist Source** → Lecture de playlists vidéo / musique
- **OBS Countdown** → Timer avancé
- **Move Transition** → Transitions fluides et avancées
- ... (d'autres à venir)

---
