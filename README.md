# OBS Plugins for Linux (Arch)

Collection personnelle de plugins OBS Studio compilés pour Arch Linux.

Je maintiens ces plugins parce que :
- Les versions officielles sont parfois lentes à arriver sur l'AUR
- Je veux des versions à jour et propres
- J'aime bidouiller et rendre les choses plus simples pour les autres Linuxiens

**Plugins disponibles :**
- Media Playlist Source (lecture de playlists vidéo/musique)
- OBS Countdown (timer avancé)
- Move Transition (transitions fluides)
- ...

Tout est open source, libre d'utilisation et compilé depuis les sources officielles.
